# 🧬 Bioinformatics & Machine Learning Portfolio (Bootcamp)

Welcome to my core Machine Learning and Computational Biology repository. 

This repository serves as a documented tracking of my intensive progression through advanced data science pipelines, algorithmic problem solving, and medical artificial intelligence modeling. As a **BSc (Hons) Biochemistry student**, I built this repository to explicitly bridge the gap between abstract biological theories and applied computational Python workflows.

## 🗂️ Repository Structure

* **`/Data`** — Contains raw numerical targets, `.fasta` genome sequences, and synthesized cancer diagnostic `.csv` files.
* **`/Scripts_Generators`** — Pure Python scripts utilizing the `os` and `random` logic to algorithmically synthesize massive biological datasets when internet queries failed.
* **`/Streamlit_App`** — The deployment architecture (`app.py` and serialized `.pkl` arrays) used to host a live, user-facing diagnostic website.

## 🧠 Core Technologies & Notebooks

This portfolio demonstrates proficiency across the following technical scopes:

1. **Scikit-Learn (Supervised Learning):** Designed classification pipelines utilizing **Random Forests**, **Logistic Regression**, and **K-Nearest Neighbors (KNN)** to diagnose synthetic breast cancer datasets (`01_Breast_Cancer_Detection.ipynb`, `The Titanic Survival Predictor.ipynb`).
2. **Mathematical Foundations (Calculus):** Manually constructed Gradient Descent algorithms from scratch to demystify complex neural network weight balancing (`Maths_Under_The_Hood.ipynb`).
3. **BioPython (Genomics):** Utilized the standard BioPython `.SeqUtils` and `.ProtParam` library logic to reverse-translate DNA strings and calculate amino acid Isoelectric/GRAVY stability scores on-the-fly (`FINAL.ipynb`, `DNA_Classifier.ipynb`).
4. **Deep Learning (TensorFlow/Keras):** Deployed a custom Convolutional Neural Network (CNN) architecture capable of performing multi-dimensional kernel scans on high-res medical imagery for real-world disease detection (`A Convolutional Neural Network.ipynb`).
